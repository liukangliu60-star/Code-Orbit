# Code-Orbit 🛸

**小米 100T Token 创造者激励计划 - 申请项目**

Code-Orbit 是一款专为小米 MiMo 生态设计的自动化研发智能体。它通过接入 MiMo-V2.5 系列模型，致力于解决复杂工程中的超长代码库审计与自动化重构。

### 🌟 核心优势
- **长文本深度感知**：专为 MiMo-V2.5 的 1M 上下文窗口设计，支持全量代码库读入。
- **Agentic Workflow**：基于“感知-规划-执行”闭环，非简单的对话助手。

### 📂 快速开始
1. 运行 `pip install -r requirements.txt`。
2. 在 `configs/mimo_config.yaml` 填入 API Key。
3. 执行 `python main.py` 开启自动化研发之旅。
